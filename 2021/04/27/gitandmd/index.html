git 分支相关

git bransh 查看本地所有分支

git branch -a产看远程分支

git push origin master 将文件给推到服务器上

git remote show origin 显示远程库 origin 里的资源

git checkout -b dev 建立一个新的本地分支 dev
git merge origin/dev 将分支 dev 与当前分支进行合并
git checkout xxx 切换到本地 xxx 分支
